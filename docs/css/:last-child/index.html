Описание

CSS псевдокласс :last-child находит любой элемент, являющийся последним в его родителе.

Синтаксис

element:last-child { style properties }

Пример

HTML

<ul>
  <li>Этот элемент не лаймовый.</li>
  <li>И этот тоже.</li>
  <li>А этот да! :)</li>
</ul>

CSS

li:last-child {
  background-color: lime;
}
