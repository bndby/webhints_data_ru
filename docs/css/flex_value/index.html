CSS тип данных <flex> определяет гибкую длину в сеточном контейнере. Он используется в таких свойствах, как grid-template-columns, grid-template-rows.

Синтаксис

Тип данных <flex> определяется как <number> с единицей измерения fr. Единица измерения fr представляет собой долю оставшегося в сетке пространства. Как и все CSS размеры, между числом и единицей измерения нет пробела.

Примеры

1fr    /* Целочисленное значение */
2.5fr  /* Дробное значение */

Спецификации

Спецификации Статус Комментарий
