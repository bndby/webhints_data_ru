--- name: getMilliseconds theme: Date lang: JS title: Date.prototype.getMilliseconds() --- ```js dateObj.getMilliseconds(); ``` --- Метод **`getMilliseconds()`** возвращает миллисекунды указанной даты по местному времени. ## Возвращаемое значение Значение, возвращённое методом `getMilliseconds()`, является целым числом от `0` до `999`. --- Пример: использование метода `getMilliseconds()` ```js var today = new Date(); var milliseconds = today.getMilliseconds(); ``` --- - [Date.prototype.getMilliseconds()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Date/getMilliseconds)
