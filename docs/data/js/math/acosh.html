--- name: acosh theme: Math lang: JS title: Math.acosh() --- ```js Math.acosh(x); ``` --- Метод **`Math.acosh()`** возвращает гиперболический арккосинус числа. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.acosh()` ```js Math.acosh(-1); // NaN Math.acosh(0); // NaN Math.acosh(0.5); // NaN Math.acosh(1); // 0 Math.acosh(2); // 1.3169578969248166 ``` --- - [Math.acosh()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/acosh)
