--- name: asin theme: Math lang: JS title: Math.asin() --- ```js Math.asin(x); ``` --- Метод **`Math.asin()`** возвращает арксинус числа (в радианах). ## Параметры **`x`** : Число. --- Пример: использование метода `Math.asin()` ```js Math.asin(-2); // NaN Math.asin(-1); // -1.5707963267948966 (-pi/2) Math.asin(0); // 0 Math.asin(0.5); // 0.5235987755982989 Math.asin(1); // 1.570796326794897 (pi/2) Math.asin(2); // NaN ``` --- - [Math.asin()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/asin)
