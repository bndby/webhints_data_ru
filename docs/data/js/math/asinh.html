--- name: asinh theme: Math lang: JS title: Math.asinh() --- ```js Math.asinh(x); ``` --- Метод **`Math.asinh()`** возвращает гиперболический арксинус числа. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.asinh()` ```js Math.asinh(1); // 0.881373587019543 Math.asinh(0); // 0 ``` --- - [Math.asinh()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/asinh)
