--- name: atanh theme: Math lang: JS title: Math.atanh() --- ```js Math.atanh(x); ``` --- Метод **`Math.atanh()`** возвращает гиперболический арктангенс числа. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.atanh()` ```js Math.atanh(-2); // NaN Math.atanh(-1); // -Infinity Math.atanh(0); // 0 Math.atanh(0.5); // 0.5493061443340548 Math.atanh(1); // Infinity Math.atanh(2); // NaN ``` --- - [Math.atanh()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/atanh)
