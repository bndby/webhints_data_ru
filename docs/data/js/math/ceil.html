--- name: ceil theme: Math lang: JS title: Math.ceil() --- ```js Math.ceil(x); ``` --- Метод **`Math.ceil()`** - округление вверх. Округляет аргумент до ближайшего большего целого. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.ceil()` ```js Math.ceil(0.95); // 1 Math.ceil(4); // 4 Math.ceil(7.004); // 8 Math.ceil(-0.95); // -0 Math.ceil(-4); // -4 Math.ceil(-7.004); // -7 ``` --- - [Math.ceil()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/ceil)
