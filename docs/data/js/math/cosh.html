--- name: cosh theme: Math lang: JS title: Math.cosh() --- ```js Math.cosh(x); ``` --- Метод **`Math.cosh()`** возвращает гиперболический косинус числа. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.cosh()` ```js Math.cosh(0); // 1 Math.cosh(1); // 1.5430806348152437 Math.cosh(-1); // 1.5430806348152437 ``` --- - [Math.cosh()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/cosh)
