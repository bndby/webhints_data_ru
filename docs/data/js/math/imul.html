--- name: imul theme: Math lang: JS title: Math.imul() --- ```js Math.imul(a, b); ``` --- Метод **`Math.imul()`** возвращает результат перемножения в стиле C двух 32-битных чисел. ## Параметры **`a`** : Первое число. **`b`** : Второе число. --- Пример: использование метода `Math.imul()` ```js Math.imul(2, 4); // 8 Math.imul(-1, 8); // -8 Math.imul(-2, -2); // 4 Math.imul(0xffffffff, 5); // -5 Math.imul(0xfffffffe, 5); // -10 ``` --- - [Math.imul()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/imul)
