--- name: log2 theme: Math lang: JS title: Math.log2() --- ```js Math.log2(x); ``` --- Метод **`Math.log2()`** возвращает двоичный (по основанию 2) логарифм числа ## Параметры **`x`** : Число. --- Пример: использование метода `Math.log2()` ```js Math.log2(3); // 1.584962500721156 Math.log2(2); // 1 Math.log2(1); // 0 Math.log2(0); // -Infinity Math.log2(-2); // NaN Math.log2(1024); // 10 ``` --- - [Math.log2()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/log2)
