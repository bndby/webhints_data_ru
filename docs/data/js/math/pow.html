--- name: pow theme: Math lang: JS title: Math.pow() --- ```js Math.pow(base, exponent); ``` --- Метод **`Math.pow()`** возвращает основание, возведённое в степень показатель, то есть, значение выражения основаниепоказатель. ## Параметры **`base`** : Основание степени. **`exponent`** : Показатель степени, в которую возводится основание `base`. --- Пример: использование метода `Math.pow()` ```js Math.pow(7, 2); // 49 ``` --- - [Math.pow()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/pow)
