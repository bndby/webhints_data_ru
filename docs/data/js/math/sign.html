--- name: sign theme: Math lang: JS title: Math.sign() --- ```js Math.sign(x); ``` --- Метод **`Math.sign()`** возвращает знак числа, указывающий на то, является ли число отрицательным, положительным или нулём. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.sign()` ```js Math.sign(3); // 1 Math.sign(-3); // -1 Math.sign("-3"); // -1 Math.sign(0); // 0 Math.sign(-0); // -0 Math.sign(NaN); // NaN Math.sign("foo"); // NaN Math.sign(); // NaN ``` --- - [Math.sign()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/sign)
