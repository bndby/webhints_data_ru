--- name: sinh theme: Math lang: JS title: Math.sinh() --- ```js Math.sinh(x); ``` --- Метод **`Math.sinh()`** возвращает гиперболический синус числа. ## Параметры **`x`** : Число. --- Пример: использование метода `Math.sinh()` ```js Math.sinh(0); // 0 Math.sinh(1); // 1.1752011936438014 ``` --- - [Math.sinh()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/sinh)
