--- name: tan theme: Math lang: JS title: Math.tan() --- ```js Math.tan(x); ``` --- Метод **`Math.tan()`** возвращает тангенс числа. ## Параметры **`x`** : Число, представляющее угол в радианах. --- Пример: использование метода `Math.tan()` ```js Math.tan(1); // 1.5574077246549023 ``` Поскольку метод `Math.tan()` принимает радианы, а чаще проще работать с градусами, следующая функция принимает значение в градусах, преобразует их в радианы и возвращает тангенс. ```js function getTanDeg(deg) { var rad = (deg * Math.PI) / 180; return Math.tan(rad); } ``` --- - [Math.tan()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math/tan)
