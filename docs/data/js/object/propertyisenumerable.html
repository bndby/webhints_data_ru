--- name: propertyIsEnumerable theme: Object lang: JS title: Object.prototype.propertyIsEnumerable() --- ```js obj.propertyIsEnumerable(prop); ``` --- Метод **`propertyIsEnumerable()`** возвращает логическое значение, указывающее, является ли указанное свойство перечисляемым. ## Параметры **`prop`** : Имя проверяемого свойства. --- ```js var o = {}; var a = []; o.prop = "перечисляемое"; a[0] = "перечисляемое"; o.propertyIsEnumerable("prop"); // вернёт true a.propertyIsEnumerable(0); // вернёт true ``` --- - [Object.prototype.propertyIsEnumerable()](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Object/propertyIsEnumerable)
