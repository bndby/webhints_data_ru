HTML-элемент <dd> (от англ. Description Details) предоставляет подробности или определение предшествующего термина (<dt>) в списке определений (<dl>).

Категории контента Нет
Разрешённое содержимое Потоковый контент
Пропуск тегов Открывающий тег обязателен. Конечный тег может быть опущен, если за элементом <dd> непосредственно следует элемент <dd> или <dt>, или если в родительском элементе нет больше содержимого.
Разрешённые родители <dl> или (в WHATWG HTML) <div> который внутри элемента <dl>.
Предыдущий элемент <dt> или другой <dd> элемент.
Разрешённые ARIA-роли Нет
DOM-интерфейс HTMLElement

Атрибуты

Этот элемент включает глобальные атрибуты.

nowrap
Если значение атрибута установлено yes, текст определения не будет переноситься. Значение по умолчанию no.
