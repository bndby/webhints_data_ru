HTML-элемент <legend> представляет собой заголовок содержания родительского элемента <fieldset>.

Категории контента (en-US) Нет
Разрешённый контент Фразовый контент (en-US).
Пропуск тегов Нет, открывающий и закрывающий теги обязательны.
Разрешённые родители Является дочерним элементом тега <fieldset> 
Разрешённые ARIA roles Нет
Взаимодействие с DOM  HTMLLegendElement (en-US)

Атрибуты

Этот элемент включает только универсальные атрибуты (en-US).

Примеры

Смотрите <form> для примеров использования элемента <legend>.
