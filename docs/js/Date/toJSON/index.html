Date.prototype.toJSON()

Сводка

Метод toJSON() возвращает представление объекта Date в виде JSON.

Синтаксис

dateObj.toJSON()

Описание

Экземпляры объекта Date ссылаются на определённый момент времени. Вызов метода toJSON() вернёт строку, отформатированную в JSON (при помощи метода toISOString()), представляющую значение объекта Date. Этот метод, как правило, предназначен для сериализации объектов Date в JSON.

Примеры

Пример: использование метода toJSON()

var jsonDate = (new Date()).toJSON();
var backToDate = new Date(jsonDate);

console.log('Сериализованный объект даты: ' + jsonDate);
