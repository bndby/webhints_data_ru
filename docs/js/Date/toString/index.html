Date.prototype.toString()

Сводка

Метод toString() возвращает строковое представление указанного объекта Date.

Синтаксис

dateObj.toString()

Параметры

Нет.

Описание

Объект Date переопределяет метод toString(), унаследованный из объекта Object; он не наследует метод Object.prototype.toString(). Для объектов Date метод toString() возвращает строковое представление объекта.

Метод toString() всегда возвращает строковое представление даты на американском английском.

JavaScript вызывает метод toString() автоматически, когда дата представляется как текст, или когда дата участвует в конкатенации строк.

Примеры

Пример: использование метода toString()

В следующем примере переменной myVar присваивается значение, возвращённое методом toString() объекта Date:

var x = new Date();
myVar = x.toString(); // присваивает переменной myVar значение вроде:
// Mon Sep 28 1998 14:36:22 GMT-0700 (Pacific Daylight Time)
