Symbol.replace

Symbol.replace — известный символ, задающий метод для замены подстрок в строке. Эта функция вызывается методом String.prototype.replace().

Смотрите также RegExp.prototype[@@replace]() (en-US) и String.prototype.replace() для получения дополнительных сведений.

Атрибуты свойстваSymbol.replace
Записываемое нет
Перечисляемое нет
Настраиваемое нет
