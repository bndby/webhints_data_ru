Хранит элементы, цель которых — помочь браузеру в работе с данными.

---

```html
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
  <head>
    <title>Заголовок</title>
  </head>
  <body>
    <h1>Заголовок</h1>
    <p>Контент</p>
  </body>
</html>
```
