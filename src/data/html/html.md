# &lt;**html**&gt;

Является контейнером, который заключает в себе всё содержимое страницы.

`xmlns`
: Определяет пространство имен XHTML-документа.

---

```html
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
  <head>
    <title>Заголовок</title>
  </head>
  <body>
    <h1>Заголовок</h1>
    <p>Контент</p>
  </body>
</html>
```
