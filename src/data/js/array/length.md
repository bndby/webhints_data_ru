# Array.length

Cвойство **`length`** объекта, который является экземпляром типа `Array`, устанавливает или возвращает число элементов этого массива.

---

```js
var items = ["shoes", "shirts", "socks", "sweaters"];
items.length;

// возвращает 4
```
