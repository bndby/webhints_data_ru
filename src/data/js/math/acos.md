title: Math.acos()

```js
Math.acos(x);
```

---

Метод **`Math.acos()`** возвращает арккосинус числа (в радианах).

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.acos()`

```js
Math.acos(-2); // NaN
Math.acos(-1); // 3.141592653589793
Math.acos(0); // 1.5707963267948966
Math.acos(0.5); // 1.0471975511965979
Math.acos(1); // 0
Math.acos(2); // NaN
```
