title: Math.atan()

```js
Math.atan(x);
```

---

Метод **`Math.atan()`** возвращает арктангенс числа (в радианах).

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.atan()`

```js
Math.atan(1); // 0.7853981633974483
Math.atan(0); // 0
```
