```js
Math.cbrt(x);
```

---

Метод **`Math.cbrt()`** возвращает кубический корень числа.

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.cbrt()`

```js
Math.cbrt(-1); // -1
Math.cbrt(0); // 0
Math.cbrt(1); // 1
Math.cbrt(2); // 1.2599210498948734
```
