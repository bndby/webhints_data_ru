```js
Math.cos(x);
```

---

Метод **`Math.cos()`** возвращает косинус числа.

## Параметры

**`x`**

: Число радиан.

---

Пример: использование метода `Math.cos()`

```js
Math.cos(0); // 1
Math.cos(1); // 0.5403023058681398

Math.cos(Math.PI); // -1
Math.cos(2 * Math.PI); // 1
```
