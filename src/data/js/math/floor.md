---

Метод **`Math.floor()`** - округление вниз. Округляет аргумент до ближайшего меньшего целого.

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.floor()`

```js
Math.floor(45.95); //  45
Math.floor(-45.95); // -46
```
