title: Math.log10()

```js
Math.log10(x);
```

---

Метод `Math.log10()` возвращает десятичный (по основанию 10) логарифм числа.

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.log10()`

```js
Math.log10(2); // 0.3010299956639812
Math.log10(1); // 0
Math.log10(0); // -Infinity
Math.log10(-2); // NaN
Math.log10(100000); // 5
```
