title: Math.min()

```js
Math.min([value1[, value2[, ...]]])
```

---

Метод **`Math.min()`** возвращает наименьшее из нуля или более чисел.

## Параметры

**`value1, value2, ...`**

: Числа.

---

Пример: использование метода `Math.min()`

```js
var x = 10,
  y = -20;
var z = Math.min(x, y);
```
