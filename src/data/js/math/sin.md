title: Math.sin()

```js
Math.sin(x);
```

---

Метод **`Math.sin()`** возвращает синус числа.

## Параметры

**`x`**

: Число радиан.

---

Пример: использование метода `Math.sin()`

```js
Math.sin(0); // 0
Math.sin(1); // 0.8414709848078965
Math.sin(Math.PI / 2); // 1
```
