title: Math.tanh()

```js
Math.tanh(x);
```

---

Метод **`Math.tanh()`** возвращает гиперболический тангенс числа.

## Параметры

**`x`**

: Число.

---

Пример: использование метода `Math.tanh()`

```js
Math.tanh(0); // 0
Math.tanh(Infinity); // 1
Math.tanh(1); // 0.7615941559557649
```
