title: String.prototype.toString()
---

```js
str.toString();
```

```ts
toString(): string; // ES5
```

---

Метод **`toString()`** возвращает строку, представляющую указанный объект.

---

```js
var x = new String("Привет, мир");
// Отобразит 'Привет, мир'
console.log(x.toString());
```
