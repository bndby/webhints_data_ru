title: String.prototype.toUpperCase()
---

```js
str.toUpperCase();
```

```ts
toUpperCase(): string; // ES5
```

---

Метод **`toUpperCase()`** возвращает значение строки, на которой он был вызван, преобразованное в верхний регистр.

---

```js
// 'АЛФАВИТ'
console.log("алфавит".toUpperCase());
```
