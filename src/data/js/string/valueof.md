Метод **`valueOf()`** возвращает примитивное значение объекта `String`.

## Синтаксис

```js
str.valueOf();
```

---

```js
var x = new String("Привет, мир");
console.log(x.valueOf()); // Отобразит 'Привет, мир'
```
